Activity 2: Active Directory: Creating Users

Layout Structure

  • Centered Title
  • Big, bold, and centered
  • Example: DHCP Server Setup Tutorial

Step by Step Guide

  1. Open Server Manager and add the Active Directory Domain Services role.
  2. Promote the server to a domain controller.
  3. Open Active Directory Users and Computers.
  4. Right-click the Users folder and select New → User.
  5. Fill in user details and finish the wizard.
