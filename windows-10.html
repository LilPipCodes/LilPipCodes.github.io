Windows 10

Installation, requirements, and tips for Windows 10.

System Requirements

  • 1 GHz CPU, 1 GB (32-bit) / 2 GB (64-bit) RAM, 20 GB disk

Installation Steps

  1. Create a bootable USB using Microsoft Media Creation Tool or Rufus.
  2. Boot and follow setup wizard, choose partitioning.

Warnings

Backup data and ensure compatible drivers for hardware.
