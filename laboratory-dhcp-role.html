DHCP Role and Joining Workstations in Domain

Configure DHCP and join client machines to your domain.

Overview

Set up DHCP scopes and verify clients receive addresses, then join domain.

Requirements / Tools

  • Windows Server with DHCP role
  • Client VM

Procedure

  1. Install DHCP role, create scopes.
  2. Authorize DHCP server in AD.
  3. Join client to the domain and test.
Notes: Avoid overlapping IP scopes in shared labs.
