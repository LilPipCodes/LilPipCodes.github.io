BIOS / UEFI Configuration

Set firmware settings for successful OS installations.

Overview

Steps to enable/disable Secure Boot, change boot order, and enable TPM.

Requirements / Tools

  • Access to system firmware (BIOS/UEFI)

Procedure

  1. Enter firmware during boot, change boot order to USB/DVD.
  2. Enable TPM and Secure Boot if required by OS.
Notes: Firmware interfaces differ between vendors.
