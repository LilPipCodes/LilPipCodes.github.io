Active Directory: Creating Users

Hands-on lab for creating and managing user accounts in Active Directory.

Overview

Learn to create user accounts, assign groups, and understand OU structures.

Requirements / Tools

  • Windows Server VM
  • Active Directory role installed

Procedure

  1. Install Active Directory Domain Services.
  2. Create an OU and add users.
  3. Assign groups and test logon.
Notes: Use strong passwords for lab accounts.
